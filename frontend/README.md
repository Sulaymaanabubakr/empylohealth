# Circles App (Frontend)

## Deployment & Builds
**IMPORTANT:** This project uses EAS Build.
Please refer to [DEPLOYMENT_NOTES.md](./DEPLOYMENT_NOTES.md) for critical information regarding signing keys, Google Sign-In configuration, and EAS credentials.

## Getting Started
-   Start dev client: `npx expo start --dev-client`
-   Run on Android: `npx expo run:android`
-   Run on iOS: `npx expo run:ios`

## Environment
- `EXPO_PUBLIC_FIREBASE_RTDB_URL=https://<your-belgium-rtdb-instance>.europe-west1.firebasedatabase.app`
